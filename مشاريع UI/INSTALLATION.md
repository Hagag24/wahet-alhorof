# دليل التثبيت - منصة الألعاب التعليمية العربية

## 📦 متطلبات النظام

قبل البدء، تأكد من أن لديك:

- **Node.js** 18.0.0 أو أحدث ([تحميل](https://nodejs.org))
- **pnpm** أو npm أو yarn (مدير الحزم)
- **محرر نصوص**: VS Code أو أي محرر آخر

### التحقق من الإصدارات:
```bash
node --version      # يجب أن يكون v18+
pnpm --version      # أو npm --version
```

---

## 🚀 خطوات التثبيت السريع

### الخطوة 1: استنساخ المشروع أو تحميل الملفات

**الخيار أ: من GitHub (إذا كان موجوداً)**
```bash
git clone https://github.com/yourrepo/educational-games-platform.git
cd educational-games-platform
```

**الخيار ب: من الملفات المُحمَّلة**
```bash
# استخرج ملفات المشروع في مجلد
cd v0-project
```

### الخطوة 2: تثبيت الحزم المطلوبة

```bash
# باستخدام pnpm (مُوصى به)
pnpm install

# أو باستخدام npm
npm install

# أو باستخدام yarn
yarn install
```

الانتظار قد يأخذ 2-5 دقائق حسب سرعة الإنترنت.

### الخطوة 3: تشغيل السيرفر

```bash
# تشغيل وضع التطوير
pnpm dev
# أو: npm run dev
```

### الخطوة 4: الدخول إلى التطبيق

افتح المتصفح واذهب إلى:
```
http://localhost:3000
```

ستظهر المنصة بجميع الشاشات الـ 10 جاهزة للاستخدام!

### تحقق من أن كل شيء يعمل:

1. **الصفحة الرئيسية تحمّل**:

   - ✅ الشخصيات تتحرك بسلاسة

## 🔧 استكشاف الأخطاء الشائعة

### المشكلة: "Port 3000 already in use"
```bash
# استخدم منفذ مختلف:
pnpm dev -- -p 3001
```

### المشكلة: خطأ في التثبيت
```bash
# احذف المجلدات المؤقتة:
rm -rf node_modules .next
pnpm install
```

### المشكلة: الحروف العربية لا تظهر
```bash
# تأكد من أن المتصفح يستخدم UTF-8
# (عادة يكون افتراضياً)
```

### المشكلة: الأداء بطيء
```bash
# استخدم وضع الإنتاج:
pnpm build
pnpm start
```

### المشكلة: TypeScript errors
```bash
# تأكد من الإصدارات:
pnpm list typescript
# يجب أن يكون 5.7 أو أحدث
```
