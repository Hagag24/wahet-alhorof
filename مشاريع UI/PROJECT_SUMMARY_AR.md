## 🛠️ البيئة التقنية

### المتطلبات
- Node.js 18+
- pnpm (مدير الحزم)

### الإطار عمل والمكتبات
- **Next.js 16.2.4**: إطار عمل React
- **React 19**: مكتبة واجهات المستخدم
- **Tailwind CSS 4**: نظام التصميم
- **TypeScript 5.7**: لغة البرمجة
- **Shadcn/UI**: مكونات جاهزة

### الأوامر الأساسية
```bash
pnpm install      # التثبيت
pnpm dev         # تشغيل التطوير
pnpm build       # البناء
pnpm start       # الإنتاج
```
