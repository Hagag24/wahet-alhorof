## 📋 نظرة سريعة

منصة تعليمية عربية **متكاملة وشاملة** للأطفال من 4-10 سنوات تجمع بين:
- 🎮 6 ألعاب تعليمية تفاعلية
- 🎤 نطق صوتي عربي كامل
- 🔊 مؤثرات صوتية احترافية
- 🎨 تصميم عصري وجميل
- 📱 متوافقة مع جميع الأجهزة

---

## 🚀 البدء السريع

```bash
# 1. تشغيل المشروع
pnpm dev

# 2. فتح المتصفح
# http://localhost:3000

# 3. استمتع!
```
