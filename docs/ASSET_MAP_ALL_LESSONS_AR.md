# خريطة صور كل الدروس لمشروع Kids

هذا الملف مخصص للتنفيذ مع Windsurf/Codex أو المصمم.

## قاعدة التصميم

- الصور الرسمية مثل جامع الأزهر يفضل أن تكون صورة حقيقية مرخصة أو صورة من العميل.
- الصور التعليمية والألعاب تُرسم بأسلوب كرتوني تعليمي.
- الشخصيات النسائية والبنات تكون محتشمة ومحجبة/مناسبة للبيئة العربية.
- لا تضع نصوصًا طويلة داخل الصورة إلا إذا كان مطلوبًا كحرف أو كلمة داخل اللعبة.
- يفضل PNG بخلفية بسيطة أو شفافة عندما يكون العنصر مستقلًا.
