# Kids All Lessons Assets Plan

هذا ZIP لا يحتوي على الصور الفعلية لكل الدروس، بل يحتوي على:
- هيكل المجلدات المطلوب داخل `public/images`.
- خريطة كاملة لكل الصور المطلوبة.
- Prompt جاهز لـ Windsurf/Codex لربط الصور بالمشروع بعد توفيرها.

الملفات المهمة:
- `docs/ASSET_MAP_ALL_LESSONS_AR.md`
- `docs/asset_map_all_lessons.json`
- `docs/WINDSURF_IMAGE_INTEGRATION_PROMPT.md`

طريقة الاستخدام:
1. وفر الصور بنفس الأسماء والمسارات الموجودة في الخريطة.
2. انسخ مجلد `public/images` داخل المشروع.
3. أعطِ Windsurf ملف البرومبت لربط الصور بالواجهات.
